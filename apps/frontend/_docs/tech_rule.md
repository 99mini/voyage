# 기술 블로그 작성을 위한 규칙

## 대원칙

### 1. 주제 의식을 버리고 소재 의식으로 쓰자

### 2. 독자 수준이 아니라 자기 수준으로 쓰자

### 3. 재미있게 글을 쓰자

## 기술 블로그의 4종류: 저, 술, 편, 집

| 구분 |                         내용                         |                          종류                           |
| :--: | :--------------------------------------------------: | :-----------------------------------------------------: |
|  저  |          직접 경험하고 실험한 과정이나 결과          |                 개발기, 도입기, 적용기                  |
|  술  |     어떤 것을 분석하여 의미를 풀이하고 해석한 것     |          기술 소개, 용어 분석, 에러 해결 방법           |
