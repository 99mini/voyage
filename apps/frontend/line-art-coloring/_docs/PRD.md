# Line Art Coloring Book

## 개요

사용자는 GPT API를 통해 생성된 라인아트(선 그림)를 받아보고, 웹에서 직접 색칠한 뒤, 완성된 이미지를 로컬에 저장할 수 있습니다.
직관적이고 쉬운 UX/UI를 제공합니다.

## 주요 기능

### 라인아트 이미지 생성

- GPT API(혹은 DALL-E 등 이미지 생성 API 활용)를 통해 사용자가 원하는 주제/스타일로 라인아트 이미지를 생성합니다.
- 생성된 이미지는 앱 내 캔버스에 표시됩니다.

### 컬러링(색칠하기) 기능

- 사용자는 브러시, 색상 팔레트, 지우개 등 기본적인 컬러링 도구를 사용해 이미지를 색칠할 수 있습니다.
- Undo/Redo 기능 제공
- 다양한 브러시 사이즈 및 투명도 조절 가능

### 저장 및 다운로드

- 사용자가 완성한 컬러링 이미지를 PNG/JPG 등 포맷으로 로컬에 저장할 수 있습니다.

### 기타

- 반응형 UI 지원(모바일/데스크탑)
- 최근 작업한 이미지 목록(히스토리) 제공 (선택)

## 기술 스택

- 프론트엔드: React, Vite, TypeScript
- 상태관리: Zustand, @tanstack/react-query
- UI/디자인 시스템: TailwindCSS, Shadcn, VDS(Voyage Design System)
- 이미지 생성: OpenAI GPT API (DALL-E 등)
- 기타: HTML5 Canvas API

## 페이지 구조 (Page Driven Development)

- / : 라인아트 생성 및 컬러링 메인 페이지
- [components]: 라인아트 생성 폼, 컬러링 캔버스, 툴바, 색상 팔레트, 다운로드 버튼 등
- [hooks]: 이미지 생성/다운로드/컬러링 관련 커스텀 훅
- [utils]: 이미지 처리, 파일 저장 등 유틸 함수

## 사용자 플로우

1. 사용자가 원하는 라인아트 주제/스타일을 입력
2. GPT API로부터 라인아트 이미지 생성
3. 캔버스에서 색칠 작업
4. 완성된 이미지를 로컬에 저장

## 요구 사항

- 빠르고 직관적인 UI/UX
- 브라우저에서 바로 사용 가능
- 외부 API Key는 환경변수로 관리
- 이미지 품질 및 저장 포맷 최적화
